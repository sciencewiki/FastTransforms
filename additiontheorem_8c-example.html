additiontheorem.c

This example confirms numerically that:

\[ f(z) = \frac{P_4(z\cdot y) - P_4(x\cdot y)}{z\cdot y - x\cdot y}, \]

is actually a degree- \(3\) polynomial on \(\mathbb{S}^2\), where \(P_4\) is the degree- \(4\) Legendre polynomial, and \(x,y,z \in \mathbb{S}^2\).

#include <fasttransforms.h>
#include <ftutilities.h>
typedef struct {
double x;
double y;
double z;
} double3;
double dot3(double3 x, double3 y) {return x.x*y.x+x.y*y.y+x.z*y.z;};
void normalize3(double3 * x) {
double nrm = sqrt(dot3(* x, * x));
x->x /= nrm; x->y /= nrm; x->z /= nrm;
}
double P4(double x) {double x2 = x*x; return ((35.0*x2-30.0)*x2+3.0)/8.0;};
double3 z(double theta, double phi) {return (double3) {sin(theta)*cos(phi), sin(theta)*sin(phi), cos(theta)};};
int main(void) {
printf("This example confirms numerically that\n");
printf("\n");
printf("\t"MAGENTA("[P‚ÇÑ(z‚ãÖy) - P‚ÇÑ(x‚ãÖy)]/(z‚ãÖy - x‚ãÖy)")",\n");

printf("Legendre polynomial, and "MAGENTA("x,y,z ‚àà ùïä¬≤")".\n");
printf("To verify, we sample the function on an "MAGENTA("N√óM")" tensor product grid\n");
printf("at equispaced points-in-angle defined by:\n");
printf("\n");
printf("\t"MAGENTA("Œ∏‚Çô = (n+1/2)œÄ/N")", for "MAGENTA("0 ‚â§ n < N")",\n");

printf("\t"MAGENTA("œÜ‚Çò = 2œÄ m/M")", for "MAGENTA("0 ‚â§ m < M")";\n");
printf("\n");

printf("the Fourier coefficients to spherical harmonic coefficients using\n");

printf("\n");
printf("In the basis of spherical harmonics, it is plain to see the\n");
printf("addition theorem in action, since "MAGENTA("P‚ÇÑ(x‚ãÖy)")" should only consist of\n");

return 0;
}
